Quick Start

Create a new post

``` bash
$ hexo new “My New Post”
```

More info: [Writing][5]

Run server

``` bash
$ hexo server
```

More info: [Server][6]

Generate static files

``` bash
$ hexo generate
```

Deploy to remote sites

``` bash
$ hexo deploy
```
